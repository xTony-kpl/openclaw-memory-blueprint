# Weekly Memory Review

## 1) Promote durable facts
- Move stable facts to `../MEMORY.md` or topic files (`preferences.md`, `projects.md`, `decisions.md`).

## 2) Rehome inactive daily notes and archive noise
- Move older canonical daily notes that should leave the active top-level window to `daily/YYYY/MM/`.
- Move long/raw logs to `archive/YYYY/`.

## 3) Remove obvious duplicates
- Keep one clean canonical version.

## 4) Safety check
- Ensure no secrets are stored in memory files.
