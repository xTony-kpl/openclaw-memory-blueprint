# Memory Index

## Long-term memory
- `../MEMORY.md`
- Purpose: stable, long-term facts only.

## Active daily notes
- `YYYY-MM-DD.md` (top level of this folder)
- Purpose: current active canonical daily notes and near-term daily context.

## Historical canonical daily notes
- `daily/YYYY/MM/YYYY-MM-DD.md`
- Purpose: older canonical daily notes after they leave the active top-level window.

## Topic memory
- `preferences.md` — stable user/assistant preferences
- `projects.md` — cross-session project state (`active` / `paused` / `completed`)
- `decisions.md` — durable decisions that should affect future work

## Archive
- `archive/YYYY/*.md`
- Purpose: raw fragments, transcripts, detailed investigations, and other long historical logs kept for trace/search.

## Workflow helper files
- `MEMORY_POLICY.md` — rules for what goes where
- `DAILY_TEMPLATE.md` — template for daily notes
- `WEEKLY_REVIEW.md` — weekly cleanup/distillation checklist
