# Decisions

## Format
- Date:
- Decision:
- Why:
- Status:
- Superseded by:

## Ordering rule
- `memory/decisions.md` is ordered by date.
- Within each date, entries are ordered by the best-supported sequence in which the decisions were actually taken, not by later insertion order.
- If exact within-day order is unclear, use the best-supported approximation and do not invent false precision.
